My Djikstra Graph Geodesic Path (MyDijkstra)

uses Dijkstra Path Geodesic Path to return shortest path between user selected points
Property Description Default(s) Restrictions
Input Set the source data set.

Accepts input of following types:

  • vtkPolyData
Selection Set the extracted selection data set.

Accepts input of following types:

  • vtkUnstructuredGrid
LineType This propery specifies which type of curvature to compute. 0

The value(s) is an enumeration of the following:

  • Geodesic (0)
  • Sulcus (1)
  • Gyrus (2)
